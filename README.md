# V Treasure Hunt

This is a FiveM script for a treasure hunt experience.

## Installation

1. Place the `V_treasurehunt` folder inside your FiveM resources directory.
2. Ensure that your server configuration (`server.cfg`) includes the resource:
   ```cfg
   ensure V_treasurehunt
   ```
3. Start or restart your FiveM server.

## Usage

- Use the in-game commands or triggers as defined in the scripts to start and participate in the treasure hunt.

## Structure

- `client/` - client-side logic
- `server/` - server-side logic
- `bridge/` - utility functions and framework interactions
- `locales/` - localization files

Feel free to modify as needed.
